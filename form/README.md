# 📋 React Comprehensive Dynamic Form

A fully interactive, controlled-component form built with **React** and styled cleanly using **Tailwind CSS**. This project demonstrates advanced form handling in React by managing a single complex state object that aggregates text inputs, dropdowns, checkboxes, and radio buttons dynamically.

---

## 🚀 Features

*   **Controlled Components:** All inputs link directly to a single consolidated piece of React state.
*   **Multi-Type Input Handling:** Clean execution covering:
    *   **Text & Email Fields:** First Name, Last Name, Email, Street Address, City, State, and Postal Code.
    *   **Dropdown Selection:** Dynamic country picker (defaults to India).
    *   **Checkboxes:** Multiple selection capabilities for email notification preferences (Comments, Candidates, Offers).
    *   **Radio Buttons:** Exclusive selection parameters for push notifications (Everything, Same as Email, No Push Notifications).
*   **State Optimization:** Utilizes a generic, scalable `changeHandler` function that handles updates via element types dynamically rather than writing separate functions for every single input.

---

## 🛠️ Tech Stack

*   **Frontend Library:** React.js
*   **State Management:** React Hooks (`useState`)
*   **Styling:** Tailwind CSS (Clean, bordered input fields and cards)
