# 🎓 Top Courses - Multi-Card Filtering App

An interactive, API-driven course filtering application built using **React** and styled with **Tailwind CSS**. This project demonstrates how to handle asynchronous data fetching, manage application loading states, use toast notifications for error handling, and perform client-side categorical filtering.

---

## 🚀 Features

* **Asynchronous Data Fetching:** Fetches dynamic course payloads from an external API endpoint on initial component mount.
* **Categorical Filtering:** Instantly filters cards based on active category states (`All`, `Development`, `Business`, `Design`, `Lifestyle`).
* **Loading Spinner Integration:** Displays a clean loading animation state while data is being resolved from the server.
* **Toast Notifications:** Integrates `react-toastify` to gracefully handle network dropouts and display errors to the user.
* **Interactive UI/UX:** Built-in hover cards complete with interactive tags, course metadata descriptions, and "Like" features.

---

## 🛠️ Tech Stack

* **Frontend Library:** React.js
* **State Management:** React Hooks (`useState`, `useEffect`)
* **Styling:** Tailwind CSS (Grid-based layout systems)
* **Notifications:** React-Toastify
