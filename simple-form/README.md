# 📝 React Simple Form Sample

A lightweight, straightforward form implementation built with **React** and styled using **Tailwind CSS**. This project serves as a clean, introductory example of managing standard input states using React's controlled components architecture.

---

## 🚀 Features

*   **Controlled State Tracking:** Synchronizes input fields seamlessly with React's internal component state.
*   **Dynamic Change Tracking:** Implements a consolidated handler function to capture and update textual states efficiently.
*   **Modern Minimalism:** Styled using clean Tailwind utility classes focusing on high readability, smooth spacing, and native borders.
*   **Console Payload Delivery:** Prevents default browser reload behavior on submit and prints a cleanly formatted data object straight to the console.

---

## 🛠️ Tech Stack

*   **Frontend Library:** React.js
*   **State Management:** React Hooks (`useState`)
*   **Styling:** Tailwind CSS
